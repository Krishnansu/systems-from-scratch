# Internet Overview

The Internet is a network of interconnected networks.

```text
                    Internet
                        │
      ┌─────────────────┼─────────────────┐
      │                 │                 │
   ISP Network      Cloud Network     Enterprise
      │                 │             Network
      │                 │                 │
   Home Router     Load Balancer      Office Router
      │                 │                 │
   Laptop        Web Server         Employee PC
```

**Key Idea:** Every network is independently managed but communicates using common protocols (TCP/IP).
