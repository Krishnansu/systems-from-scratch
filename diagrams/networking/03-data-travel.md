# How Data Travels

```text
Application
      │
      ▼
Operating System
      │
      ▼
Network Interface Card
      │
      ▼
Cable / Wi-Fi
      │
      ▼
Router
      │
      ▼
Internet
      │
      ▼
Destination Router
      │
      ▼
Destination NIC
      │
      ▼
Operating System
      │
      ▼
Application
```

Every message follows this general journey.
