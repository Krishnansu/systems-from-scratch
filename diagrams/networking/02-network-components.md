# Components of a Network

```text
+-----------+      Wi-Fi / Ethernet      +-----------+
| Laptop A  |  <---------------------->  | Laptop B  |
+-----------+                            +-----------+
       │                                       │
       └──────────── Communication ────────────┘

Nodes
↑
Devices participating in communication
```

A network consists of:

- Nodes (devices)
- Communication medium
- Protocols
